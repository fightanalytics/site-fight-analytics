# Site Fight Analytics #

Address: [http://fightanalytics.cc](http://fightanalytics.cc)

### How do I set up? ###



#### * Dependencies ####

* [Jekyll](https://jekyllrb.com/docs/installation/)

#### * Configuration ####

1. Clone the repo;
2. Make your changes under the `gh-pages` branch;
2. Run `jekyll serve`.

#### * Deployment ####

1. Run `git push origin gh-pages`.
